# RL_DOOM_AGENT
Using DRL to train an RL Doom Agent
### Libraries needed:
  1. `vizdoom`.
  2. `numpy`.
  3. `gymnasium`.
  4. `torch`.
  5. `stable_baselines3`.
  6. `torchvision`.
  7. `torchaudio`.
  8. clone the vizdoom reposetory **https://github.com/Farama-Foundation/ViZDoom.git**.
